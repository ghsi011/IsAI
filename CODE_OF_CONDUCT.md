# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, be constructive, assume good faith, and keep the
discussion about the work. Harassment, personal attacks, and discriminatory
behavior are not tolerated.

Additionally, given this project's subject matter: discussions must not promote
academic dishonesty. IsAI exists to help humans review and improve writing —
requests or contributions aimed at evading institutional review processes are
off-topic and will be moderated.

Report unacceptable behavior via a private GitHub security advisory or to the
repository maintainer.
